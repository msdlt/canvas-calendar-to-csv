Canvas Calendar to CSV

Follow these instructions to extract events from your existing Canvas calendar and convert them into a CSV file

  1. Make sure you are logged into Canvas in this browser window.
  5. Go to this Canvas API Calendar Link, which will open in a new tab/window.
  6. Copy and paste the entire contents of that page into the box below, then click the "Convert" button.

  7. This will generate a CSV file for you to save/download.
  8. Open Excel and go to Data > From Text, then select the file.
  9. In Text Import Wizard:
    1. Select "Delimited", then "Next".
    2. Select "Other" only and type ~ in box, then "Finish".
    3. Make sure it is inserting in cell A1 and click "OK".
  10. Check and edit the events as required, then save as CSV and import using the Calendar Import tool in the appropriate course in Canvas.

IMPORTANT: If you are uploading events for a single section, e.g. a Medicine Year 5 firm, please make sure you select the correct Section from the "Section" dropdown in the Calendar Import tool. Otherwise, you will import the events for all students in the course, so they will all have their calendars cluttered up with events that are not for them. If you do this by mistake, you can delete the incorrect import using the bin button for that import, then try again.
